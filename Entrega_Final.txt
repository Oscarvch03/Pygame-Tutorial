Lenguaje de Programación de Desarrollo del Proyecto: Python


Descripción del Problema Y Solución Propuesta: “Tutorial PyGame”
La idea que se tiene para el Proyecto es crear un repositorio en GitHub 
en el cual se implementen algunos ejemplos base acerca de la librería 
“PyGame” de Python, en el cual se logre explicar de forma clara y concisa 
sobre algunas funciones que se usan en los ejemplos (tomando como 
referencia el libro “How to Think Like a Computer Sciencist” el cual 
se encuentra en el siguiente link: 
http://www.ict.ru.ac.za/Resources/cspw/thinkcspy3/thinkcspy3.pdf), 
para que las personas que se interesen en la Programación Orientada a 
Objetos y desarrollo de videojuegos en 2D se familiaricen con esta.
Para esto es importante aclarar las diferencias que tiene la Programación 
Dirigida por Eventos en contraste con Programación Procedural, y Programación 
Orientada a Objetos.
Adicionalmente se implementara un programa basado en el juego clásico conocido 
como “Pacman” en el cual se recapitule y se utilice como base lo anteriormente 
aprendido en los ejemplos base.

Funcionalidad y Descripción de la herramienta:
Link del Repositorio en GitHub: 
https://github.com/Oscarvch03/Pygame-Tutorial


En el anterior link, se encuentra todo el Proyecto subido, el cual incluye: 
    1)	Un instructivo que muestra paso a paso como instalar la librería Pygame, 
        ya que esta no viene por defecto al instalar Python.
    2)	Carpeta: Ejemplos de implementaciones básicas con Pygame
    3)	Carpeta: Ejemplos de implementaciones de Eventos con la librería “turtle”.
    4)	Ambas carpetas cuentan un archivo Descripcion.txt en el cual se hace una

        carpetas. Cada ejemplo cuenta con comentarios guía los cuales ayudan al 
        usuario a entender con mayor facilidad el código implementado (haciendo que 
        dichos comentarios cumplan la función de Docstrings o Documentación de cada 
        ejemplo)
    5)	Las Entregas de Avance semana a semana.
    6)	La Entrega Final.
    7)	Para nuestro ejemplo principal decidimos implementar un programa basado en 
        el juego clásico conocido como “Pacman”  y anexo se encuentra su documentación.
